DummyJS

The absolute quickest and easiest way to write and run functional tests for websites.

Install

DummyJS is a command line tool and can be installed with git and npm.

$ git clone https://github.com/mennovanslooten/DummyJS.git
$ npm install DummyJS -g

More about installing DummyJS

Write

Tests are simple lists of commands and asserts.

open             http://my.test.site
click            .show-greeting
assertVisible    .greeting

More about writing DummyJS tests

Run

Tests are run from the command line in a headless browser.

$ dummyjs mytest.dummy

############################################################
# Starting ./mytest.dummy (4 actions)
############################################################

✓ open                    http://my.test.site
✓ click                   .show-greeting
✓ assertVisible           .greeting

PASS: Executed 3 actions in 1s.
