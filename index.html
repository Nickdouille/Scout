DummyJS

Smart Functional Testing

Write E2E tests in English:

open            http://my.test.site
type            :text                 John
click           "Say hi!"
assertText      .greeting             Hi, John!

Run from a command line:

$ dummyjs mytest.dummy

✓ open                http://my.test.site
✓ type                :text                 John
✓ click               "Say hi!"
✓ assertText          .greeting             Hi, John!

PASS: Executed 4 actions in 1s.

DummyJS will automatically:

  • Run the tests in a real (invisible) browser.
  • Trigger all mouse, keyboard and scroll events a user would.
  • Deal with delays like page loads, Ajax requests and animations.

DummyJS tests are:

Robust

Tests are very resilient to changes in the page. Elements can change, move or disappear and DummyJS will adapt.

Easy to write

Instructions are plain English. Target elements by their text content or with CSS selectors.

Easy to run

DummyJS is a command line tool with various options to make your life easier.

Fast

Because DummyJS automatically deals with delays, tests don't need artificial waitFor instructions.
